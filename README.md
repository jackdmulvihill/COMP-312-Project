## Apache Ant
- **To compile, run, and test this application, use Apache Ant**
- https://ant.apache.org/

## To Compile New Changes
- **ant clean (Optional, but recommended)**
- **ant compile**

## To Run
- **ant run**

## To Test
- **ant compile-test (If testing suite is modified)**
- **ant test**
